# suss-calender-exporter
Export Course Timetable from SUSS e-services onto your calender

Log in to SUSS e-services -> Course Timetable

Copy the timetable with the headers.

E.g.
S/N	Course Code	Course Group	Semester Type	Date	Day	Time (From)	Time (To)	Venue	Remarks
1	ICT162	TG01	Regular	12 JAN 2026	MONDAY	12:00 PM	03:00 PM	-	[Lesson is recorded for audit, learning, and attendance-management purposes.] Venue to be confirmed
2	ICT233	TG01	Regular	13 JAN 2026	TUESDAY	12:00 PM	03:00 PM	-	[Lesson is recorded for audit, learning, and attendance-management purposes.] Venue to be confirmed
3	ICT259	TG01	Regular	15 JAN 2026	THURSDAY	12:00 PM	03:00 PM	-	[Lesson is recorded for audit, learning, and attendance-management purposes.] Venue to be confirmed
4	NCO102	TG08	Regular	16 JAN 2026	FRIDAY	08:30 AM	11:30 AM	HQ BLK C - SR.C.8.10	[Lesson is recorded for audit, learning, and attendance-management purposes.]

Paste it onto the website -> Preview -> Download .ics

Import the .ics file to the calender of your choice.

Tip: Import into a seperate calender, so it's easier to remove and reupload in the event of changes to the timetable
